# Signal Anomaly Detection using Machine Learning

## Overview
This project demonstrates how signal processing and machine learning
can be combined to detect abnormal behavior in signal data.

## Methodology
- Signal generation and preprocessing
- Feature extraction (RMS, Energy, etc.)
- Training a machine learning model (Random Forest)
- Evaluating performance using confusion matrix and feature importance

## Tools & Libraries
- Python
- NumPy
- SciPy
- scikit-learn
- Matplotlib

## Results
- Successfully detected abnormal signals
- Model performance evaluated using accuracy and confusion matrix

## Applications
- Sensor fault detection
- Condition monitoring
- Predictive maintenance
